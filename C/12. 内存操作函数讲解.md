#  2. memcpy介绍🚩
memcpy的函数原型:
![在这里插入图片描述](https://i-blog.csdnimg.cn/blog_migrate/7ea299e3342556340d29e7dd55493069.png)
<font color = red>**功能:从source的起始位置复制num个字节到destination的内存起始位置**</font></br>

用法示例:

```c
#include<stdio.h>
#include<string.h>
int main()
{
	int a[5] = { 1,2,3,4,5 };
	int b[3] = { 33,22,11 };
	memcpy(a, b, sizeof(int) * 3);//将数组b往后12个字节(3个整型)的内容拷贝到数组a
	for (int i = 0; i < 5; i++)
	{
		printf("%d ", a[i]);
	}
	return 0;
}
```

#  3. memmove函数介绍🚩
**memmove的函数原型:**
![在这里插入图片描述](https://i-blog.csdnimg.cn/blog_migrate/51fc5306f1705f2eca3dc85f25de95ba.png)
<font color = red>功能:从source的起始位置复制num个字节到destination的内存起始位置</font>

>***memmove和memcpy函数的功能是一样的,只不过memmove函数的source和destination指向的空间可以重叠***

==**用法示例:**==

```c
#include<stdio.h>
#include<string.h>
int main()
{
	int a[9] = { 1,2,3,4,5,6,7,8,9 };
	memmove(a, a + 5, 16);

	for (int i = 0; i < 9; i++)
	{
		printf("%d ", a[i]);
	}
	return 0;
}
```

**这个结果符合我们的预期:
(覆盖了前面的1234后,后面的56789照常打印)**
![在这里插入图片描述](https://i-blog.csdnimg.cn/blog_migrate/fc24f627d075f80f8bb8db8b713c2250.png)

##  3.1 memmove的缺陷🏁
- <font color = blue>memmove函数无法使用某些机器所提供的特殊字节-字符串处理指令来实现</font>
👍 👍  👍
- <font color = blue>导致的结果:memmove会比memcpy的效率低一些</font>
👍 👍 👍
- <font color = blue>如果源和目标参数真的可能存在重叠,就应该使用memmove!</font>

#  4. memset函数介绍🚩
memset的函数原型:
![在这里插入图片描述](https://i-blog.csdnimg.cn/blog_migrate/3a7626557c1861ce3ba9a51342de5efe.png)

<font color = red>功能:把从ptr开始的num个字节的值都设置为value</font>

==**用法示例:**==

```c
#include<stdio.h>
#include<string.h>
int main()
{
	char a[6] = { 'a','b','c','d','e','f'};
	memset(a, '-', 3);

	for (int i = 0; i < 6; i++)
	{
		printf("%c ", a[i]);
	}
	return 0;
}
```

**结果和我们预期的一样:
(将前三个字节的内容改成 ' - ' )**

![在这里插入图片描述](https://i-blog.csdnimg.cn/blog_migrate/31c34c52a306169803882a0904206e63.png)
><font color = orange>注意,这里内存操作函数中参数num的单位都是字节,如果你想要修改一个整型数组的前n个值,别忘了用n乘以整型的长度!</font>

#  5. memcmp函数介绍🚩
memcmp的函数原型:
![在这里插入图片描述](https://i-blog.csdnimg.cn/blog_migrate/50f83d2d48ef302793b8f3f5d70a78d7.png)

<font color = red>功能:memcmp对两段内存ptr1和ptr2的内容进行比较，共比较num个字节</font>

==**用法示例:**==

```c
#include <stdio.h>
#include <string.h>
int main ()
{
  char buffer1[] = "DWgaOtP12df0";
  char buffer2[] = "DWGAOTP12DF0";
  int n = memcmp( buffer1, buffer2, sizeof(char)*5);
  return 0;
}
```
**memcmp的返回类型和strcmp一样:**

- <font color = purple>负数代表ptr1小于ptr2.</font>
- <font color = purple>正数代表ptr1大于ptr2.</font>
- <font color = purple>0代表它们两个相同</font>

><font color = blue>值得注意的是:这些值是按照无符号字符逐字节比较,所以它用于比较不是单字节数据如整数,浮点数时可能给出不可预料的结果. </font>

#  6. memchr函数介绍🚩
**memchr的函数原型:**
![在这里插入图片描述](https://i-blog.csdnimg.cn/blog_migrate/06010139646e44a3ffd3de7b84d7698a.png)

==**功能:**==
- <font color = red>从ptr开始,查找value第一次出现的位置</font>
- <font color = red>共查找num个字节.</font>
- <font color = red>如果找到,返回一个指向该位置的指针</font>
- <font color = red>如果没找到,返回NULL.</font>

==**用法示例:**==

```c
#include<stdio.h>
#include<string.h>
int main()
{
	char a[6] = { 'a','b','c','d','e','f'};
	char* p = memchr(a, 'd', sizeof(char) * 6);//在数组a中查找字符d第一次出现的位置
	for (int i = 0; i < 3; i++)//这里返回一个指针指向字符d,并将这个指针赋值给p
	{
		printf("%c ", p[i]);
	}
	return 0;
}
```
